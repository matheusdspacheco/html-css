O que são pseudo-classes

Você aprendeu que para criar-mos seletores de IDs utilizamos (#) e para seletores de classes utilizamos (.). Para seletores de pseudo-classes utilizaremos (:).

As pseudo classes devem estar relacionadas a um elemento ou uma classe, sendo relacionados ao estado de um elemento.

Usando a pseudo-classe hover

Existe uma pseudo-classe chamada hover, uma pseudo classe de vários elementos e significa que quando o mouse for posicionado em cima do elemento ele apresentará um comportamento diferente. Escondendo um elemento da página

Como nós gostamos de mistério, não queremos apenas mudar a cor do texto, queremos também exibir um texto que esta oculto.

Revelando um elemento escondido

Revelando um parágrafo, podemos criar o seletor div > p para modificar somete o parágrafo que estiver dentro de uma div e adicionar a propriedade display: none; para ocultar tal parágrafo. Em seguida, crie o seletor div:hover > p oe adicione a propriedade display:block para exibir o texto. Por diversão configure também uma cor diferente para exibição, um background color e um width de 30px.
