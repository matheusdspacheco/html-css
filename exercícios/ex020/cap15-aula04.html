Pseudo-elementos em CSS

Ainda discutindo o assunto da aula anterior, vamos hoje estudar os pseudo-elementos.

Criando um novo exercício

Nosso exemplo de hoje não será no mesmo arquivo das anotções. No botão "Acessar Exemplo" você poderá acessar um arquivo chamado "links.html".

Criando uma lista com links

Nosso objetivo é personalizar três links: um para o repositório do GitHub, um para o canal do Youtube e um para o site do Curso em Vídeo.

Por padrão, links que já foram acessados possuem cor roxa, já os que anda não foram visitados possuem coloração azul.

Criando os estilos

Criada a lista, vamos criar nossos estilos.

Inicialmente altere a font-family do body para uma fonte não serifada. De maneira mais específica para os link altere a cor para vermelho (alterando dessa forma para sites visitados e não visitados). Tire o sublinhado dos links com text-decoration: none;. Finalize deixando os links em negrito.

Para configurar o comportamento dos links acessados, digite a:visited e configure a cor ara um vermelho escuro.

Vamos também utilizar a pseudo-classe hover para sublinhar um link quando o mouse estiver posicionado em cima dele com text-decoration: underline; e de cor preta.

Pseudo-classe active

Essa psuedo classe é muito interessante poisé por meio dela que configuraremos comprtamento de um elemento no exato mometo em que estivermos clicando nele. Quando deixarmos de clicar ele retornará ao estado anterior. Para exempliicar configure tal estado para que tenha uma cor amarela.

Pseudo-elementos

Os pseudo-elementos podem agir em classes, elementos e ids. Em suma, eles mexe no conteúdo periférico do elemento. Todo pseudo elemento é antecedido de (::).

Para exemplificar poemos indicar ao final de cada link que aquilo é um link. Para isso vamos utilizar o pseudo eleento after, seguido da propriedade content: link;, sendo possível configurar um esilo diferente para essa informação.

Como podemos ver, é possível adicionar coisas nas regiões periféricas de algum elemento sem alterar nada no HTML.

Adicionando conteúdos extras

Se você procurar por "Símbolor HTML5" e acessar um site chamado erikasarti poderá personalizar ainda mais adicionando símbolos no lugar do texto "link". para adicionar ao início do elemento utilize o pseudo elemento before.

Links especiais com formatação por classe

Se, por exemplo, você desejar indicar que o mais importante é o link para o canal no Youtube você pode criar uma classe com nome "especial" e repetir os dois pseudo elementos utilizados anteriormente, configurando també outras formatações diferenciadas
