Usando o ID com CSS

Agora mais do que nunca é importante que você tenha assistido as aulas anteriores. Se surgirem muitas dúvidas, revise.

Na aula de hoje vamos estudar sobre seletores personalizados e quando utilizálos.

Anteriormente vimos que é possível realizar uma configuração para cada elemento do HTML. Ocorre, porém, que muitas vezes temos dois elementos da mesma classe em HTML mas desejamos que eles tenham formatações diferentes. Isso é possível utilizando o ID.

Para exemplificar, vamos utilizar as próprias anotações dessa aula para criar uma página de exemplo.

Criando um estilo externo para a página

Diferente dos exemplos criados até então, hoje nosso CSS será externo.

Crie a regra charset, em body configure o background color e configure a fonte. Façamos o processo Análogo para o seletores h1 e h2.

Por padrão, todo seletor é genérico

Perceba que configurando o seletor h1 todos os respectivos títulos sofreção as alterações. É possível diferenciar de forma semantica, utilizando a hierarquida de títulos mas podemos também realizar isso utilizando inteiramente CSS.

Identificando componentes HTML com id

Temos nesse momento o título da aula e os tópicos com o mesmo nível de título. Para segregar o título principal podemos digitar dentro da tag de abertura o identificador id='principal'. Apesar de serem a mesma tag, eles estão separados pela ID.

O uso do símbolo cerquilha #

Nossa atual configuração está aplicada para todos os títulos. Para aplicar uma configuração diferente a nosso título 'principal' deveremos criar um seletor com a identificação precedido pela tag (na verdade isso é opcional) e pela cerquilha, vulgo "jogo da velha", ficando da seguinte forma: h2#principal. Nesse momento, qualquer configuração aplicada nesse seletor será aplicado somente a esse título, além de sobrepor a ele qualquer configuração diferente que esteja no seletor genérico.

Qualquer coisa pode ser configurada com esse DI "principal'. Dessa forma é interessante preceder a cerquilha pela tag.

Uma dica interessante, até mesmo para adiantar o proximo conteúdo é o seguinte:

  • Tudo o que em HTML for ID, em CSS será #.
  • TUdo o que em HTML for class, em CSS ser "." (ponto).

Personalizando os títulos com id

Vamos agora configurar um background-color em nosso "principal', adicionar um padding e um border-radius

Cuidado com erros de digitação

Fique sempre muito atento a forma como você escrevel o nome de seu ID.
