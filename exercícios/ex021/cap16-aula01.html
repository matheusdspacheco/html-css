CRIANDO O PRIMEIRO EXERCÍCIO

Antes de tudo vaos criar um pequeno exercício. Inicialmente, crie o exercício "ex021" e o nosso exemlo de Modelo de Caixas.

A prmeira coisa que faremos é criar um h1 que é uma caixa box-level. Nesse sítulo escreva Exemplo de caixa box-level". Abaixo, crie um parágrafo e tenha certeza que a estrutura tenha a seguine forma:

<h1>Exemplo de caixa box-level</h1>
<p>Parágrafo também é um exemplo de caixa box-level, mas os <a href="">links são exemplos de caixa inline-level</a>. Vamos ver como isso funciona.</p>

USANDO O DEVTOOLS DO CHROME

Para entender na prática, pelo google chrome, selecione a opção inspecionar. Acessando a aba elementos podemos verificar o modelo de caixas.

Basicamente, quando clicamosem um elemento podemos vesificar as configurações de CSS criadas automaticamente pelo useragente, vulgo, seu navegador. Todo navegador possui uma configuração base. Em nosso exemplo, nosso h1 e nosso parágrafo são do tipo display: block;, indicando ser um elemento do tipo box-level. Essa característica pode ter seu valor alterado, inclusive para flex, uma tecnologia nova da CSS. Se selecionarmos o nosso link, não será exibido nada, mas é um inline-box.

IDENTIFICANDO PADDING E MARGIN

Posicionando o mouse em cima de um conteúdo veremos apenas sua caixa. Posicionando o muose em cima do elemento podemos ver também a margem do elemento. Se o elemento tivesse borda ou "padding" seria também possível visualizá-los.

CRIANDO UM ESTILO PERSONALIZADO

Você está costumado a configurar o CSS de se exemplo antes mesmo do início, certo?Hoje ainda não fiemos nada de CSS em nosso exemplo, mas vamos resolver isso rapidamente.

Antes de qualquer estilo mais complexo vamos configurar nossa página. Adicione um background color em nosso h1 para verificar o quanto de nossa tela será ocupada pela caixa. Vamos alterar altura e largura de nosso h1 para gerar umquadrado de 300px.

A coisa mais interessante do DevTools é quevocê pode configurar vários desses elementos direto no navegador, claro que não salvará essas alterações, mas nos permitirão testar tais alterações. Brinque um pouco com com isso. É também possível desligar algumas propriedades das CSS.
