SOMBRA NAS CAIXAS

Com a base do modelo de caixas, podemos comçar a estudar as sombras que podemos aplicar nelas.
A base desse conteúdo é importante pois utilizaremos o mesmo exercício da aula anterior.

EXERCÍCIO DA AULA ANTERIOR

Na aula anterior montamos um pequeno modelo de site. Até o momento parece que apenas colocamos pedaços de papel em cima de uma mesa. Mesmo que seja fino, ainda existirá uma sombra.

CUIDADO COM AS SOMBRAS

Não é necessário exagerar. O cérebro irá perceber a sombra.

RESOLVENDO O PROBLEMA COM O RODAPÉ

Antes de aplicarmos as sombras, vamos resolver um pequeno problema.
Nosso rodapé foi configurado com um padding de 1px, entretnto você percebeu que o padding aplicado não corresponde a 1px "nem aqui e nem na China". Quando colocamos 0px o padding simplesmente some. Isso ocorre nesse caso pois quando colocamos o padding, como o texto do rodapé está em um parágrafo, ele está considerando 1px após a margem do parágrafo que está com 16px. Para resolver esse problema, configure o parágrafo "filho" do footer com margem de 0px.

ENTENDENDO A SHORTHAND BOX-SHADOW

As sombras são por padrão configuradas com uma shorthand. É curioso pois, ao contrário das configurações vistas anteriormente a shorthand para sombras não é uma simplificação da configuração.

Adicionaremos uma sombra a nosso menu, ou seja, na caixa <nav>.

Dentro do box-shadow, devemos informar alguns parâmetros. Os principais são, respectivamente, deslocamento horizontal, delocamento vertical, espalhamento (blur) e cor da sombra. Feito isso, poderemos ver uma pequena sombra em nosso nav.

USANDO O DEVTOOLS PARA AJUDAR COM AS SOMBRAS

Selecionando o box-shadow podemos variar o valor dos parâmetros para melhor entendermos.

ASSISTENTE SÍMPLES PARA A CRIAÇÃO DAS SOMBRAS

Acerca do espalhamento, quanto maior o número, mais presente na tela ela estará.

Podemos também criar nossa própria sombra. Para isso,a primeira coisa será remover o box-shadow. Agora, no DevTools, clique em adicionar box-shadow, você poderá configurar de maneira bem dinâmica. Imediatamente após o blur é posssível configurar mais um parâmetro chamado visualização. Por fim, basa adicionar a configuração em seu CSS. É também possível adicionar uma sombra interna no elemento, bastandoapenas adicionar o parâmetro "inset" antes do deslocamento horizontal.

CONFIGURANDO A COR DE UMA SOMBRA

Mudar cor de sombra é ridiculo pois não existe sombra colorida. O correto é sempre criar uma sombra prea e reduzir a opacidade da cor.

Vale lembrar uma coisa: nao se deve adicionar sombra de caixa em texto. para criar ma sobra do texto você deve utilizar o text-shadow, mas isso é assunto para um futuro breve.
