Usando Fontes Externas Baixadas

Conforme ométodo visto na aula anterior podemos adicionar infinitas fostes ao nosso site.Ocorre porém que muitas vezes o cliente precisa de apenas uma fonte e essa fonte não está o google fonts.

Criando um novo exercício

Crie o "fonte02", ou essa página de anotações.

Montando o estilo CSS

Configure seu body com uma fonte não serifada e seu h1 com uma fonte serifada e tamanho de 3em.

Sites para baixar fontes

Muitas vezes o cliente pode não gostar da fonte padrão, pedindo alguma fonte específica ou enviando-lhe um arquivo de fonte.

Para realizar o download de fontes você pode acessar sites como o Dafont e Befonts.

Cuidado com fontes sem acentos

Tome cuidado pois muitas fontes não possuem glifos com acentos. Fontes sem glifos acentuados não funcionam para textos em português.

Baixando uma fonte

Vamos escolher a fonte Love Larry. É uma fonte qua não possui diferença entre letras maiúsculas e minúsculas. Descompactando o arquivo .zip obteremos um arquivo ttf. Muitas fontes fornecem os formatos otf e ttf.

Onde colocar uma fonte externa no site

Vamos agora colar esse arquivo na pasta de nosso exercício.

Criando o seletor font-face

Para que possamos adicionar tal fonte é necessário adicionar a regra, um seletor especial chamado @font-face no exato mesmo local onde adicionariamos a regra @import para carregar as fontes do Google Fonts.

Ao adicionarmos tal seletor deverão ser exibidas as propriedades font-family onde digitaremos uma safe combination com o nome da fonte e src onde indicaremos o arquivo da fonte.

A escolha entre formato OTF ou TTF vai depender do navegador pois alguns respondem melhor a determinados formatos que outros. No geral você pode colocar os dois.

Configurando a fonte personalizada

Na URL sdo source vamos digitar a seguinte informação:

url('love larry ttf.ttf') format('').

Entenda sobre os tipos e formatos de fonte

Dentro do format podemos informar os seguintes formatos:

  • Opentype

    • OTF

  • Truetype

    • TTF

  • embedded-opentype

  • Truetype-aat

    • Apple Advanced Typography

  • SVG

Nesse caso vomos optar por truetype. Caso vá adicionar outros formatos basta adicionar uma vírgula e repetir a estrutura da url indicando o outro formato.

Configurações adicionais ao font-face

Podemos configuar nosso font-weight e font-style como normal pois não há variações nessa fonte.

Podemos agora adicionar um nome para essa font-family que não necessariamente precisa ser o mesmo nome da fonte. Na verdade você pode dar o nome que você quiser. Para poupar tempo de digitação vamos apenas chama-la de 'Love'

Por fim, configure essa fonte em seu h1.

Um bizu é indicar o font-weight como normal para garantir que não ocorrerá uma tentativa de colocar tal fonte em alguma variação, mesmo se tivermos indicado isso no font-face.

Configurando pastas específicas para fontes

É recomendado nunca somente largar a fonte na pasta principal. O ideal é sempre criar uma pasta específica para isso. Criando uma pasta e colocando a fonte lá dentro é necessário utilizar o velho comando Ctrl+Espaço para indicar o caminho da fonte. Lembre-se que agora as áspas não devem ser usadas, a menos que você digite o caminho manualmente ('fonts/love larry ttf.ttf).
