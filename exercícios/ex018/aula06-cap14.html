Acessando o Google Fonts

Utilizando as fontes padrão ficamos muito limitados pois não conseguimos utilizar toas as configurações possíveis de peso e estilo das fontes. Para resolver esses ploblemas existe um site chamado Google Fonts. Nesse repositório temos muitas famílias de fontes disponíveis, entre elas a Work Sans, uma fonte variável e uma das mais complexas existentes.

Filtrando fontes por categoria

Podemos organizar as fontes por categorias e utilizá-las em nosso site. Inicialmente vamos exibir somente a categoria Handwriting para nosso título, selecionando a fonte Kaushan Script.

Usando uma Fonte Escolhida

Você pode realizar o download da fonte, ou pode também não baixar nada. Basicamente você pode clicar em "select" e será aberta uma janela lateral onde serão exibidos nossos códigos de incorporação.

@import url('https://fonts.googleapis.com/css2?family=Kaushan+Script&display=swap');

É importante dizer que por ser uma fonte extraida de um repositório por meio de um API é necessário estar conectado a internet para que a mesma seja exibida.

Código de Incorporação da Fonte

Você pode inserir a fonte tanto por meio de um link do HTML quanto por meio das CSS, como feito na aula anterior.

Como essa fonte possui apenas o estilo regular, o link da api será pequeno.

Colocando o Código no Arquivo CSS

Copiando o import, iremos adicionar esse código imediatamente abaixo da teg de abertura Style

Uma dica interessante é que é possível importar todas as fontes que você desejar utilizar por meio de uma única API. Basta você selecionar todas elas no Google Fonts sem remover nenhuma antes de adicionar outras.

Escolhendo uma Fonte Mais Completa

Vamos novamente adicionar a fonte Work Sans.

Na tela principal você pode digitar no Google Fonts um texto de exemplo para analisar o resultado.

Importando a Segunda Conte

Removendo a fonte anterior, selecione todos os estilos da Work Sans e importe abaixo da Kaushan Script. Se não quiser utilizar todas, selecione apenas as que desejar.

@import url('https://fonts.googleapis.com/css2?family=Kaushan+Script&family=Work+Sans:ital,wght@0,100;0,
200;0,300;0,400;0,500;0,600;0,700;0,800;0,900;1,100;1,
200;1,300;1,400;1,500;1,600;1,700;1,800;1,900&display=swap');

Aplicando a Fonte ao Conteúdo HTML

No google fonts também será exibido uma safe combination para uso no CSS. Altere o estilo no Body e no H1 conforme vimos em aulas anteriores. Em h1 configure um tamanho de 3em.

No caso de um h2, configure também a fonte Kaushan.

Lembre-se que para cada fonte que você for importar é necessário importar a fonte e que essa API deverá sempre vir antes de qualquer marcador.

Mais Opções do Google Fonts

Existem outras categorias de fontes no Google Fonts, podendo ser classificadas pelo estilo, pelo idioma e pelo número de estilos.

E se a Fonte Não Estiver no Google Fonts

Existem outros sites, mas o Google Fonts é o melhor.

Quantas Fontes Usar em um Site?

Não exagere muito. Normalmente utiliza-se 3 ou 4 famílias de fontes, separando entre "grandes títulos", "destaque" e "texto normal".
