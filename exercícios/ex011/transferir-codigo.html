Para que apareça o player é necessário adicionar o parâmetro controls.
Essa maneira é um pouco limitada pois. Em alguns navegadores nem mesmo irá funcionar o autoplay.

Uma forma de não termos tanta limitação é adicionando os sources dentro do par de tags audio sem os parâmetros do exemplo anterior.

Podemos adicionar o parâmetro preload que contém três valores possíveis:

  • auto
    Valor padrão, o site só irá carregar completamente após o arregamento completo do áudio. Isso é perigoso pois, dependendo do tamanho do áudio, o site irá demorar muito para carregar.
  • metadata
    Só irá carregar informações de nome do arquivo, data, duração e alguns outros dados
  • none
    Só irá carregar o áudio quando o usuário clicar em play

Como dito anteriormente, a ordem das tags é muito importante. O navegador tentará executar cada tag na sequência até que encontre uma compatibilidade entre formato de arquivo e navegador. No nosso caso adicionamos um parágrafo com um texto de erro e um link para download.

Um bizu: apesar da melhor qualidade, jamais use WAV como arquivo para exibição durante uma navegação pois é extremamente pesado.
